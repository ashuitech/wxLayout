## 快速入门

### 第一步：启动应用

```bash
npm install
npm run dev
```

应用将在浏览器中自动打开 http://localhost:3000

### 第二步：输入内容

在左侧的Markdown编辑器中输入或粘贴您的文章内容。

### 第三步：选择主题

在底部工具栏选择一个适合您文章风格的主题。

### 第四步：复制到微信公众号

点击右侧预览区域上方的"复制到剪贴板"按钮，然后粘贴到微信公众号编辑器中。

## 基本操作

### 输入内容

1. 在左侧编辑器中直接输入
2. 从其他Markdown编辑器复制粘贴
3. 支持拖拽文件导入（未来版本）

### 切换主题

1. 在底部工具栏点击主题按钮
2. 预览区域立即更新
3. 所有内容样式同步切换

### 调整样式

#### 强调色
- 点击颜色选择器选择颜色
- 或直接输入16进制色值（如 #FF5733）
- 影响步骤说明、对比分析等元素

#### 段落间距
- **紧凑**：适合信息密集型内容
- **标准**：适合大多数文章
- **宽松**：适合文艺类内容

#### 字号
- **小**：14px，适合长篇文章
- **中**：16px，标准阅读尺寸
- **大**：18px，适合移动端阅读

### 复制到微信公众号

1. 点击"复制到剪贴板"按钮
2. 打开微信公众号编辑器
3. 粘贴（Ctrl+V）
4. 格式完全保留，无需调整

### 下载HTML文件

1. 点击"下载"按钮
2. 选择保存位置
3. 生成HTML文件，可用于其他用途

## 常见问题

### Q1：复制到微信公众号后格式乱了怎么办？

**A**：确保使用"复制到剪贴板"按钮，不要直接复制预览区域的HTML代码。如果仍有问题，尝试刷新页面重新生成。

### Q2：代码块没有语法高亮怎么办？

**A**：检查代码块是否正确指定了语言标识，如 \`\`\`javascript。如果语言不支持，会显示为纯文本。

### Q3：主题切换后样式没有更新？

**A**：刷新页面或重新生成HTML。如果问题持续，清除浏览器缓存后重试。

### Q4：如何自定义主题？

**A**：当前版本不支持自定义主题，但可以通过调整强调色、段落间距、字号来微调样式。

### Q5：支持哪些浏览器？

**A**：支持Chrome 80+、Firefox 75+、Edge 80+、Safari 13+等现代浏览器。

### Q6：文章很长，处理速度慢怎么办？

**A**：工具经过优化，1000字文章处理时间<3秒。如果文章特别长，建议分段处理。

### Q7：生成的HTML代码体积太大怎么办？

**A**：工具已优化HTML体积，膨胀率<150%。如果仍嫌大，可以尝试使用更简洁的主题。

### Q8：如何导入现有的Markdown文件？

**A**：当前版本支持复制粘贴。未来版本将支持文件导入功能。

## 最佳实践

### 内容组织

1. **结构清晰**：合理使用标题层级
2. **段落分明**：适当使用空行分隔段落
3. **重点突出**：使用粗体、引用强调重要内容
4. **代码规范**：代码块指定语言，便于高亮

### 样式选择

1. **主题匹配**：根据内容类型选择合适主题
2. **间距适中**：大多数情况使用标准间距
3. **字号合理**：根据目标读者选择字号
4. **色彩协调**：强调色与主题风格协调

### 性能优化

1. **避免过长文章**：建议单篇文章不超过5000字
2. **减少图片数量**：图片会影响加载速度
3. **简化表格**：复杂表格影响渲染性能
4. **合理使用代码块**：大量代码块影响处理速度

### 微信公众号适配

1. **测试预览**：复制前在预览区检查效果
2. **移动端测试**：在手机上预览最终效果
3. **图片优化**：使用合适尺寸的图片
4. **链接测试**：确保链接可以正常打开
